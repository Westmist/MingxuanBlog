Centos中安装MySQL
数据库
数据库
发布日期:   2020-03-17
作者:   明轩
阅读次数:  

环境简介

  • 系统镜像：Centos 7.2
  • MySQL 版本：8.0.20 MySQL Community

卸载自带的mariadb-lib

# rpm -qa | grep mariadb
mariadb-libs-libs-5.5.52-1.el7.x86_64
# rpm -e --nodeps mariadb-libs-libs-5.5.52-1.el7.x86_64

下载MySQL

官网下载对应版本的 MySQL

将 rpm 文件上传至服务器

安装依赖

# yum install glibc.i686
# yum list libstdc++*

安装 rpm 包

进入rpm包路径进行本地安装

# yum localinstall  ./xxxxxxxx.rpm

检查安装是否成功

# yum repolist enabled | grep "mysql.*-community.*"

成功则显示MySQL的各个组件

安装MySQL服务

# yum install mysql-community-server

安装过程还需要下载额外的依赖，腾讯云小水管等了一下午 (*+﹏+*)~@

若安装报错 Error :Delta RPMs disabled because /usr/bin/applydeltarpm not installed. 则还需安装如下的包 

# yum provides '*/applydeltarpm'
# yum install deltarpm

登录尝试

# mysql -u root

报错 ERROR 2002 (HY000): Can’t connect to local MySQL server through socket '/var/lib/mysql/mysql.sock' (2)

原因是 /var/lib/mysql 的访问权限问题, 更改目录的所有者即可解决

查询默认密码

# grep "temporary password" /var/log/mysqld.log

默认密码为12位随机字符

如查询不到，则可能是上次安装有残留 (重装镜像居然还有残留). 清除残留，重启服务

# rm -rf /var/lib/mysql
# systemctl restart mysqld

实在不行删库跑路，也挺省心！ ╮(￣▽ ￣)╭

使用默认密码登录并更改密码

# mysql -u root -p '6;w:C6asJak.'
mysql> ALTER USER 'root'@'localhost' IDENTIFIED BY 'YourNewPassword';

至此就可以愉快的使用MySQL了！≧◇≦
