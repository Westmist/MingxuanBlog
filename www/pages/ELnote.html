一、EL简介

表达式语言（Expression Language），或称EL表达式，简称EL，是Java中的一种特殊的通用编程语言，借鉴于JavaScript和XPath。主要作用是在Java Web应用程序嵌入到网页（如JSP）中，用以访问页面的上下文以及不同作用域中的对象 ，取得对象属性的值，或执行简单的运算或判断操作。(引用维基百科的定义)

二、EL语法

${EL表达式}

三、禁用EL

要使用EL Web服务器必须支持Servlet2.4/jsp2.0;为了对低版本环境兼容可禁用EL

  • 禁用单个EL表达式 \${EL表达式}

  • 使用page指令禁用 <%@ page isELLgnored=true %>

  • 配置web.xml进行禁用

<jsp-config>
    <jsp-property-group>
        <!--映射匹配所有jsp页面，即在所有jsp页面中禁用EL-->
        <url-pattern>*.jsp</url-pattern>
        <el-ignored>true</el-ignored>
    </jsp-property-group>
</jsp-config>

四、运算符

EL支持算术运算符、关系运算符、逻辑运算符、条件运算符(三目运算符)和empty运算符

EL中的 '+' 不能实现java中的拼接操作，如果在两个字符串中使用 '+' 会将抛出NumberFormatException

五、EL的隐含对象

1.上下文对象

访问jsp内置对象中的request对象 ${pageContext.request}

EL可以直接访问request的属性值,在JSP网页中显示str的字面量：

<%
  String str = "Hello Expression Language";
  request.setAttribute("flag",str);
%>
${flag}

利用request对象获取远程请求的IP地址使用request.getRemoteAddr(),在EL中只需去掉get将首字母改为小写即可

${pageContext.request.remoteAddr}

访问response对象,获取响应的内容类型，这里结果为"text/html;charset=UTF-8"

${pageContext.response.contextType}

访问 out 对象,获取缓冲区大小

${pageContext.out.bufferSize}

访问session对象，获取session的有效时间

${pageContext.session.maxInactiveInterval}

访问page对象，获取当前页面类文件

${pageContext.page.class}

2.作用域范围的隐含对象

pageScope隐含对象用于返回page范围内的属性值的合集，返回值为java.util.Map对象

JavaBean中的内容：

package com.lcc.javabean;
public class UserInfo {
    private String name;
    public void setName(String name){
        this.name = name;
    }
    public String getName(){
        return name;
    }
}

创建page范围内的JavaBean实例：

<jsp:useBean id="user" scope="page" class="">
     <jsp:setProperty name="user" property="name" value="Hello" />
</jsp:useBean>
<!-- 获取JavaBean的属性值 -->
${pageScope.user.name}

requestScope隐含对象用于返回request范围内的属性值的集合

<%
  String str = "Hello";
  request.setAttribute("flag",str);
%>
<!-- 在网页输出str的字面量 -->
${requestScope.flag}

sessionScope、applicationScope等隐含对象与requestScope对象的用法相同

3.访问环境信息的隐含对象

param用于获取请求参数的值

<input name="name" type="text">
<!-- 使用EL获取输入框的值 -->
${param.name}

paramValues获取参数名对应的多个值

编码<input name="live" type="checkbox" value="coding">
阅读<input name="live" type="checkbox" value="reading">
运动<input name="live" type="checkbox" value="sport">
<!-- 在网页中显示live的值 -->
${param.live[0]}
${param.live[1]}
${param.live[2]}
  • header---->headerValues
  • initParam---->initParamValues

    • 用法与param对象的用法相同

    Cookie对象

    Cookie cookie = new Cookie("user","Lcc");
response.addCookie(cookie);
<!-- 获取Cookie的值 -->
${cookie.user.values}

    六、定义和使用EL函数

    在jsp中使用EL函数需要导入依赖 jstl.jar 和 standard.jar

    1. 在WEB-INF目录中创建JSP Tag Library Descriptor文件(标签库描述文件)

    2. jsp中引入jstl支持 <%@taglib prefix="fn" uri="/stringDeal" %>

    将换行和空格进行Html转义

    在src中创建EL函数的类(普通的java类):

    import com.elfun.StringDeal;
public class StringDeal {
    //创建EL函数，必须为public修饰的静态函数
    public static String shiftEnter(String str){
        String newstr = str.replaceAll("\r\n","<br>");
        newstr = newstr.replaceAll(" ","&nbsp;");
        return newstr;
    }
}

    转义前的JSP页面：

    <form name="form" method="get" action="deal.jsp">
<textarea name="field" rows="9" cols="70"></textarea>
<br>
<input name="button" type="submit" value="提交">
</form>

    转义后的JSP页面：

    <%@ taglib prefix="fn" uri="/stringDeal" %>
<%requset.setCharacterEncoding("UTF-8");%>
<html>
${fn:shiftEnter(param.field)}
</html>
