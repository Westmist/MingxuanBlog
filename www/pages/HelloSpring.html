一、Spring概念

Spring 框架是 Java 平台的一个开源的全栈（Full-stack）应用程序框架和控制反转容器实现.

二、Spring 框架结构

Spring 是模块化的，允许你挑选和选择适用于你的模块，不必要把剩余部分也引入.

三、导入依赖

在pom.xml文件中配置依赖的Maven坐标(SSM框架所需依赖)

四、Spring 的使用

创建 Bean 类
public class HelloWorld {
   private String message;
   public void setMessage(String message){
      this.message  = message;
   }
   public void sayMessage(){
      System.out.println("Your Message : " + message);
   }
}
Spring Bean 配置

在resources目录下创建 Spring 配置文件 Beans.xml，添加如下的 Bean 配置

<bean id="HelloSpring" class="com.study.HelloSpring">
    <property name="messages" value="Hello Spring!"></property>
</bean>
在 main 方法中使用 Bean
// 创建一个Spring的IOC容器对象
ApplicationContext context = new ClassPathXmlApplicationContext("Beans.xml");
// 从IOC容器中获取Bean实例
HelloSpring obj = (HelloSpring) context.getBean("HelloSpring");
// 调用sayMessages()方法
obj.sayMessages();
编译执行

在终端输出Hello Spring!

五、总结

控制反转：上例中对象的创建不是通过 new 方式实现，而是交给 spring 配置创建类对象.

依赖注入：依赖的对象没有通过手动调用 setXX 方法去设置，而是通过配置赋值.(配置 Beans.xml 中的 bean)
